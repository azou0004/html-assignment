HTML Tags I Learned

<h1> to <h6>
Heading tags used to define headings from largest (h1) to smallest (h6).
<p>
Paragraph tag used for blocks of text.
<ul> <ol>
List tags: unordered (bullets) and ordered (numbers).
<a>
Anchor tag used to create hyperlinks to other pages or websites.
<img>
Image tag used to display images on a web page.
